Фитнесс-клуб с рассылкой уведомлений
================

После установки и сборки пакетов yarn, композера и миграций:

```
$ composer install
```
```
$ yarn install
```
```
$ symfony run yarn encore dev
```
```
$ symfony console doctrine:migrations:migrate
```
